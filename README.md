# Sistema de Registro de Asistencia - Centro de Cómputo ITCM

Versión Estática (Compatible con Vercel/GitHub Pages).

## Características

- **Registro de Entrada y Salida**
- **Historial de Asistencia**
- **Almacenamiento Local**: Los datos se guardan en el navegador (LocalStorage). NO requiere servidor ni base de datos.
- **Exportación a CSV**

## Instalación y Uso

Simplemente abre el archivo `index.html` en tu navegador.

## Despliegue en Vercel

1.  Sube este repositorio a GitHub.
2.  Importa el proyecto en Vercel.
3.  Vercel detectará automáticamente que es un sitio estático.
4.  ¡Listo!

## Nota Importante

Al usar `LocalStorage`, los registros de asistencia solo son visibles en el dispositivo donde se crearon. Si cambias de computadora, los datos no estarán ahí.
